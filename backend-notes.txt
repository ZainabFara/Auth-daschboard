1. Dessa paket är olika vertyg för backend:
express: webbramverk för att skapa API och webbservrar.
cookie-parser: för att hantera cookies i mina HTTP-förfrågningar.
mailttrap: för att testa e-postutskick lokalt
bcryptjs: för atta hasj löserord
dotenv: För att hantera miljövariabler (konfigurationer som API-nycklar och hemligheter).
jsonwebtoken: För att hantera JWT (för autentisering och säkerhet).
mongoose: För att interagera med MongoDB-databasen

2. kör npm i nodemon -D sen npm run dev
