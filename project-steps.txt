1. Skapa en mapp och klona GitHub-repo:
jag bröjade med att skapa en mapp och klona ett GitHub-repo. OBS! Glöm in fixa ssh key för att kunna pusha arbetet 

2. Skapade en package.json genom att köra npm init -y.
Den här filen används för att hålla reda på projektets metadata och alla beroenden som används i projektet,
vilket gör det enkelt att återinstallera alla nödvädniga paket om projektet överförs till en annan dator. Vilket 
innebär att package.json inte bara håller reda på installerade paket utan även deras versioner och scripts för att köra projektet.

3. Installerade sen de nödvändiga paket för projektet alltså för backend-delen:
npm install express cookie-parser mailtrap bcryptjs dotenv jsonwebtoken mongoose

4. Setup backend med index.js och starta servern:
jag skapade filen index.js i backend-mapp och satte upp en enkel server med hjälp av express:

5. package.json:
Här använder du import för att hämta express. Notera att import är en ES6-funktion som kräver att du specificerar "type": "module" i din package.json
ändrade namnet från start till dev och skrev in detta : nodemon backend/index.js
